The phone outline + browser frame are pure SVG with fine strokes and gold corner accents. Each one wraps an autoplaying muted video that pauses when out of viewport.

The video compression was the tricky part — ffmpeg wouldn't install via Homebrew on this Intel Mac (Tier 2 config), so I used the npm `@ffmpeg-installer/ffmpeg` binary instead. 37MB → 4.9MB, no quality loss noticeable at the mockup size.
